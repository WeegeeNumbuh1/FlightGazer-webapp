Tips & Tricks

Expand/Collapse
  • Turning off the screen at night
    • Configure Settings ‚Üí Brightness

    Enable Dynamic Brightness and set Secondary Brightness to 0
    If you don't want it to turn off at sunset, disable Use Sunrise/Sunset then set Brightness Switch Time.
    Note: FlightGazer will still be running *and* driving the screen, even with a brightness of 0.

  • Only turn on the screen when there's an aircraft nearby (no clock)
    • Configure Settings ‚Üí Brightness

    Set Brightness to 0 and disable Dynamic Brightness.
    Enable and set Active Plane Display Brightness to the desired value.
    Note: If you use NO_FILTER mode and follow this tip, the display will remain blank.

  • Hiding elements on the display
    • Configure Settings ‚Üí Colors

    Simply set the display element to 'BLACK'

  • Extending the life of the physical display
    • The display is a wear item.
    Over time, some of the pixels may become dimmer or their ability to reproduce accurate color is hindered.
    To delay this effect, lower the brightness of the display.
